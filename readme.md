Introduction

A security audit was performed to check the computer’s security settings. The goal was to ensure the system is protected from threats such as malware, hackers, and unauthorized access.


Security Checks Performed
1️⃣ Check Password Strength

First we check if the computer uses a strong password.

Steps:

Press Ctrl + Alt + Delete

Click Change a password

Check if the password is strong

A strong password should have:

At least 8 characters

Uppercase letters (A B C)

Lowercase letters (a b c)

Numbers (1 2 3)

Symbols (! @ #)

Example of a strong password:

Kgomo@2026!

2️⃣ Check Firewall Status

A firewall protects the computer from hackers and unsafe networks.

Steps:

Click the Start button

Type Windows Security

Open Windows Security

Click Firewall & network protection

Check that the following are ON:

Domain Network → ON

Private Network → ON

Public Network → ON

3️⃣ Check Windows Updates

Updates fix security vulnerabilities.

Steps:

Click Start

Click Settings

Click Windows Update

Click Check for updates

Install updates if available

4️⃣ Run a Virus Scan

This checks the system for malware or viruses.

Steps:

Open Windows Security

Click Virus & threat protection

Click Quick Scan

Wait for the scan to finish.

5️⃣ Check Startup Programs

Too many startup applications can slow down the computer and create security risks.

Steps:

Right-click Start

Click Task Manager

Click the Startup tab

Disable unnecessary programs

Examples:

Spotify

Skype

Teams

✅ Conclusion

The security audit confirmed that important system security features such as firewall protection, system updates, and malware scanning are active. Maintaining strong passwords, updating software regularly, and managing startup programs helps keep the system secure.
